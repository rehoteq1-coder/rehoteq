Step 1 — Prepare: Download SP Flash Tool v5.2152 or v6.x. Extract to folder. Do NOT run as admin yet.
Step 2 — Driver: Install MTK VCOM USB driver. Check Device Manager — "MTK USB Port" must appear. If driver fails, use Zadig to force-install WinUSB.
Step 3 — Scatter File: Open SP Flash Tool → Click "Scatter-loading" → Browse to your ROM folder → Select MT6xxx_Android_scatter.txt
Step 4 — Flash Mode: Select "Download Only" for standard flash. Select "Firmware Upgrade" if phone is bricked. NEVER use "Format All" unless you know what you're doing.
Step 5 — Flash: Click Download button (green arrow). POWER OFF phone completely. Connect USB — phone auto-enters download mode. Wait for green ring.
Step 6 — Complete: Green ring appears = success. Disconnect phone. Remove battery 5 seconds. Boot up normally (first boot may take 5-10 mins).
📦 Odin — Samsung
Used for: All Samsung Galaxy phones and tablets
Step 1: Download Odin 3.13.1 or 3.14.4 (most stable). Download correct firmware from SamFirm, SamMobile or Frija — match exact model number.
Step 2 — Download Mode: Power off Samsung. Hold Vol Down + Home + Power (old) or Vol Down + Vol Up while plugging USB (new models). Accept warning.
Step 3 — Odin Setup: Run Odin as Administrator. ID:COM port must show blue (phone detected). If not — install Samsung USB driver.
Step 4 — Load Files: Extract firmware .zip. Load: AP → AP_xxx.tar.md5 | BL → BL_xxx | CP → CP_xxx | CSC → CSC_xxx (use HOME_CSC to keep data or CSC for full wipe)
Step 5 — Options: Enable "Auto Reboot" and "F. Reset Time". DO NOT check Re-Partition unless scatter file issue. Click Start.
Step 6: PASS (green) = success. Phone auto-reboots. First boot 5-15 mins. If FAIL — check firmware matches exact model and CSC region.
📦 Mi Flash Tool — Xiaomi / Redmi / POCO
Used for: All Xiaomi, Redmi, POCO, Black Shark phones
Step 1: Download Mi Flash Tool from Xiaomi official site. Install Mi USB driver. Download correct MIUI fastboot ROM (.tgz) for exact model from MIUI firmware site.
Step 2 — EDL/Fastboot Mode: Power off. Hold Vol Down + Power for Fastboot. For EDL mode (bricked): Vol Up + Vol Down while connecting USB or use testpoint.
Step 3: Open Mi Flash Tool. Click "Select" → browse to extracted ROM folder. Phone must show in device list.
Step 4 — Flash Type: "Clean All" = full wipe (recommended for bootloop). "Save User Data" = keeps data. "Clean All and Lock" = factory state with locked bootloader.
Step 5: Click Flash. Progress bar fills to 100%. Phone reboots automatically. First MIUI boot can take 8-15 minutes.
📦 iTunes / 3uTools — iPhone / iPad
DFU Mode (iPhone 8+): Press Vol Up → Press Vol Down → Hold Side button 10 sec → Release Side, hold Vol Down 5 sec. Screen stays black = DFU success.
DFU Mode (iPhone 7): Hold Vol Down + Sleep/Wake 10 sec → Release Sleep/Wake, keep Vol Down 5 sec.
Recovery Mode (all): Force restart until recovery screen shows. iTunes/Finder detects automatically.
iTunes Restore: Device shown in iTunes → Restore iPhone (full wipe + latest iOS) or Update (keeps data). Hold Shift+Restore to browse to specific .ipsw file.
3uTools: Better than iTunes for technicians. Flash → Real Flash → browse .ipsw. Supports downgrade with signed firmware. Shows detailed device info.
📦 ADB / Fastboot Commands
Works on all Android phones with unlocked bootloader
Essential Commands — tap to copy:
adb devices — Check if device is recognized
adb reboot bootloader — Enter fastboot mode
adb reboot recovery — Boot into recovery
fastboot oem unlock — Unlock bootloader (wipes data!)
fastboot flash recovery recovery.img — Flash custom recovery
adb sideload update.zip — Sideload OTA/ROM via recovery
fastboot -w — Wipe userdata (factory reset via fastboot)
adb shell wm density reset — Fix display/UI issues
adb shell pm uninstall -k --user 0 — Remove bloatware without root
